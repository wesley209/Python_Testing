70 Convenience function. Connect to *address* (a 2-tuple ``(host,
71 port)``) and return the socket object. Passing the optional
72 *timeout* parameter will set the timeout on the socket instance
73 before attempting to connect. If no *timeout* is supplied, the
74 global default timeout setting returned by
75 :func:`getdefaulttimeout` is used. If *source_address* is set it
76 must be a tuple of (host, port) for the socket to bind as a source
77 address before making the connection. A host of '' or port 0 tells
78 the OS to use the default.
79  
80 .. versionchanged:: 20.6.0

83 :func:`socket.inet_pton`.
