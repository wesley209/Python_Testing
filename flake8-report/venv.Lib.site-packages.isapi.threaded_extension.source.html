57 # A generic thread-pool based extension, using IO Completion Ports.
58 # Sub-classes can override one method to implement a simple extension, or
59 # may leverage the CompletionPort to queue their own requests, and implement a
60 # fully asynch extension.

130 def Dispatch(self, ecb):
131 """Overridden by the sub-class to handle connection requests.
132  
133 This class creates a thread-pool using a Windows completion port,
134 and dispatches requests via this port. Sub-classes can generally
135 implement each connection request using blocking reads and writes, and
136 the thread-pool will still provide decent response to the end user.
137  
138 The sub-class can set a max_workers attribute (default is 20). Note
139 that this generally does *not* mean 20 threads will all be concurrently
140 running, via the magic of Windows completion ports.
141  
142 There is no default implementation - sub-classes must implement this.
143 """
144 raise NotImplementedError("sub-classes should override Dispatch")
