459 """
460 "Shut-down the queue, making queue gets and puts raise
461 `ShutDown`.

463 By default, gets will only raise once the queue is empty. Set
464 *immediate* to True to make gets raise immediately instead.
465  
466 All blocked callers of `put` and `get` will be unblocked.
