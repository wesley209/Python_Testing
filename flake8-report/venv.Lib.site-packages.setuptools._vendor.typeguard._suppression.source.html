33 """
34 Temporarily suppress all type checking.
35  
36 This function has two operating modes, based on how it's used:
37  
38 #. as a context manager (``with suppress_type_checks(): ...``)
39 #. as a decorator (``@suppress_type_checks``)
40  
41 When used as a context manager, :func:`check_type` and any automatically
42 instrumented functions skip the actual type checking. These context managers can be
43 nested.
44  
45 When used as a decorator, all type checking is suppressed while the function is
46 running.
47  
48 Type checking will resume once no more context managers are active and no decorated
49 functions are running.
50  
51 Both operating modes are thread-safe.
52  
53 """
