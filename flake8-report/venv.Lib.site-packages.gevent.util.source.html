43 Because every exception that is unhandled by greenlet will be logged,
44 it is desirable to prevent non-error exceptions from leaving a greenlet.
45 This can done with a simple ``try/except`` construct::
46

48 try:
49 return func(*args, **kwargs)
50 except (TypeError, ValueError, AttributeError) as ex:

54  
55 wrapped_func = wrap_errors((TypeError, ValueError, AttributeError), func)
